# Ruby合宿2023春・team2「spring」

本プログラムはRuby合宿2023春において2班が作成を行ったパズルゲームです（完成形には至らず…）。


# ゲーム内容

プレイヤーキャラクターを←キーと→キーで操作。↑キーでジャンプをして、爆弾を避けながら上へ登っていこう。 

ただし、途中にある色のついたコインをとると、その色の足場しか踏めなくなるため、注意して登っていくこと（未実装）。


# 操作方法

main.rb実行でゲームスタート！

・←キー：キャラクターを左へ動かせる。
・→キー：キャラクターを右へ動かせる。
・↑キー：キャラクターをジャンプさせられる。滞空中に←キーや→キーでコントロール可能。
